# Getting Started
Instructions to install and get up and running with **agamotto**.
{: .fs-6 .fw-300}

**agamotto** has been tested on the following:

- Ubuntu 20.04:
    - Local server, without Docker
    - Local server, with Docker
- Windows 11:
    - Local server, without Docker
